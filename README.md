    ## Возможности

    - Автоматическая разметка в GPT  
    - 4 стандартных раздела Windows  
    - Для **Windows To Go** и чистых установок  
    - Один `.cmd` файл — ничего больше не нужно  
    - Работает на Windows 10/11 (x64 и ARM64)

    ---

    ## Структура разделов

    | Раздел       | Размер     | ФС     | Назначение                  |
    |--------------|:----------:|:------:|-----------------------------|
    | **EFI**      | 300 МБ     | FAT32  | Загрузчик UEFI              |
    | **MSR**      | 16 МБ      | —      | Зарезервировано Microsoft   |
    | **Windows**  | Остаток    | NTFS   | Основной раздел             |
    | **Recovery** | 800 МБ     | NTFS   | Восстановление              |

    ---

    ### Схема
    +--------------------------------------------------------------------------+
    | EFI (300МБ) │ MSR (16МБ) │ Windows (остальное) │ Recovery (800МБ)       |
    +--------------------------------------------------------------------------+
    text---

    ## Как использовать

    1. Скачай: [`create_win_partitions.cmd`](./create_win_partitions.cmd)  
    2. **Правый клик → Запуск от имени администратора**  
    3. Введи номер диска → жди
    =========================================
    Разметка завершена успешно!
    text### Пример вывода

    ```cmd
    ====================================================
    СКРИПТ РАЗМЕТКИ ДИСКА ПОД WINDOWS
    ====================================================
    Диск 0    SSD 512 ГБ
    Диск 1    USB 32 ГБ
    Введите номер диска: 1
    Создание EFI... OK
    Создание MSR... OK
    Создание Windows... OK
    Создание Recovery... OK
    =========================================
    ГОТОВО!
    =========================================

    Настройка размеров
    Открой скрипт и измени:
    cmdcreate partition efi size=300
    shrink desired=800

    Примечания

    Использует только встроенный DiskPart
    Работает на флешках и внутренних дисках
    Идеально для Windows To Go
    Добавлены паузы для 100% стабильности
